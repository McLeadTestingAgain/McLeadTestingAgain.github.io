Starting a Cascading Style Sheet

There are a few ways to apply CSS within an HTML document. You can create the CSS on a separate file and then link in the CSS into the html document. You can place the CSS directly into a HTML tag. However, for today's lesson we are going to create your CSS by embedding our CSS into the head of our HTML document. Under our title tags within our HTML document we will begin our CSS by typing an opening style tag:

< style type = "text/CSS">

Our CSS also needs to be closed. In order to close our CSS we need to use a closing style tag:

< /style >

CSS Syntax

Between our opening and closing style tags we will type our CSS. CSS looks differently than HTML, for it does not rely on the use of tags. Rather CSS has its own syntax; which will look similar to those of us who are familiar with C++ or Java. CSS consists of rules which tell the web-browser how and what to do with the CSS. The rules of CSS consist of three parts. The parts are:

  • Selector - tells the web browser where to apply the rule.
  • Property - tells the web browser what attribute of the html tag will be controlled by the rule.
  • Value - assigns a value to the property within the rule.

For instance in the code below all of the text h3 (header size 3) tags will be changed to blue.

The selector is followed by an opening curly brace { within the opening curly brace and the closing curly brace } the rules to apply to the selector are listed. These rules collectively are known as the rule set. Some things to keep in mind about the syntax of CSS:

  • Each rule has at a property and at least one value.
  • Each property and value(s) within the rule is separated by a colon :
  • Individual rules are separated by each other by a semi-colon ;
  • If a property or value has more than one word in its name, than it is hyphenated for instance below for font-weight the name is hyphenated, for instance.
    • h2
    {
    font-weight: bold;
    }
  • You can have more than one selector for rule set, just separate the selectors with a comma, for instance.
    • h1, h3
    {
    color: red;
    }
